## Project Webut
ふと浮かんだ独り言やメモを、そっと放り投げて気軽にシェアできる場です。誰かと絡む必要もなく、ただのんびり、あなたのメモに共感の『いいね』が届くだけ。心地よく、自分のために、ほのぼのと過ごせるSNS離れの空間をお届けします。

## はじめる

### 依存関係のインストール
```
npm i
```

### 環境変数の構築
```
DIRECT_URL= " "
DATABASE_URL=" "
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
NEXT_PUBLIC_Google_Client_ID=
```

### マイグレーションの同期
```
npx prisma generate
```

### 実行
```
npm run dev
```
